三角函數微分公式
sin(x)' = cos(x) tan(x)' = sec²(x) sec(x)' = sec(x)tan(x)
cos(x)' = -sin(x) cot(x)' = -csc²(x) csc(x)' = -csc(x)cot(x)
[sin⁻¹(x)]' = 1/√1-x² [tan⁻¹(x)]' = 1/1+x² [sec⁻¹(x)]' = 1/x√x²-1
極限公式
lim(x->0) sin(x) / x = 1 lim(x->0) tan(x) / x = 1
lim(x->0) e^(x)-1 / x = 1 lim(x->0) (1-cos(x))/x² = 1/2
lim(x->0) sin⁻¹(x) / x = 1 lim(x->0) tan⁻¹(x) / x = 1
